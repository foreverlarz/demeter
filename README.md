# demeter
Automated embedded watering system for outdoor plants.

**Feature status:**
* Wifi connectivity: working
* NTP sync: working
* Timekeeping: working
* Manual watering: working
* Automatic watering: unimplemented
* Weather-dependent watering: unimplemented

**Parts list:**
